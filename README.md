# Go Helloworld Project

A simple Go Lang Hello World application.

## Prerequisites

- Go 1.21 or higher installed on your machine
- [Download Go](https://golang.org/dl/)

## Running the Project

### Option 1: Run directly
```bash
go run main.go
```

### Option 2: Build and execute
```bash
go build -o helloworld
./helloworld
```

## Expected Output
```
Hello, World!
```

## Project Structure
- `main.go` - Main application file with entry point
- `go.mod` - Go module definition file
- `README.md` - Project documentation
